# 🛍️ Zay eCommerce Demo

A simple and responsive eCommerce demo website built with HTML, CSS, Bootstrap and JavaScript.

🔗 **Live Demo:**  
https://amirgheisarnezhad.github.io/zay-ecommerce-demo/

---

## 📌 Features

- Responsive layout using Bootstrap
- Navigation between multiple pages
- Simple contact form validation with JavaScript
- Basic login simulation (frontend only)
- Image slider (carousel)

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- Bootstrap 5  
- JavaScript
